# RAISE Application Workspace

Vue3 CDN + SFC + UnoCSS demo console for the RAISE the STAKES
application package.

Run locally:

```powershell
node scripts\static-server.mjs 4178
```

Open:

```text
http://127.0.0.1:4178/
```

The first screen is the usable product workspace for ContestOps AI, focused on
the RAISE application packet.
